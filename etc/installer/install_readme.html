Welcome to jDip

Overview

jDip is a an adjudicator and mapping program for Diplomacy™-based games with an advanced graphical interface for entering and viewing orders. jDip even supports entering orders in text form, from email, and importing entire games from Judges or http://www.floc.net.

The fast automated adjudicator is very accurate, and supports the latest (2000) rules. More than 23 variants are supported.
