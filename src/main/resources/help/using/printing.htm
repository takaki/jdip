Printing

Maps may be printed from within jDip. Printing is WYSIWYG; the same map that is on the screen is printed, but scaled to fit the paper.

To print a map from jDip:

  1. Set the view options in the View menu (e.g., if you do not wish to show orders for a particular power)

  2. Select Print... from the File menu

  3. Set the desired print options.
