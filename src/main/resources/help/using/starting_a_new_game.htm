Starting A New Game

To start a new game, select New | Game... from the File menu. The New Game window will then appear, with several tabs to set the type of game and options for that game.

Variant Selection

A list of available variants is displayed on the left. When a variant is selected, its description will appear to the right, and available maps (some variants have more than one map choice) will appear at the lower-left.

Some variants have multiple maps. The map selection is not permanent; you may select a different map once the game has been created, by selecting Select Map... from the View menu

Map and Symbols

This tab allows for the selection of the map style and game symbol styles. Note that some variants may only have one map; other variants may have several from which to choose.

Should you wish to change the map style or units after the game has started, this may be done from the View | Map and Symbols... menu

Starting Options

The starting phase may be set here as well as the victory conditions.

The Reset Options button will reset all Starting Options to their default values.

Rule Options

These are options that may be specific to a given variant. The list of available options is shown at left. When an option is selected (by a click), a brief description of and available choices for that option are shown on the right.

By holding the mouse over an option choice, a more lengthy description of that choice will be displayed.

The Reset Rules button will reset all Rule Options to their default values.
