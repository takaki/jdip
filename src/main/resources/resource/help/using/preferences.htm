Preferences

Preferences control various options within jDip. Preferences are saved and recalled automatically. Options are arranged on different tabs within the Preferences window.

General

Restore last window size and position at restart

If selected, jDip will remember the last window size and position, and when restarted, use these settings.

Show resolution results after orders are resolved

If selected, show the Adjudication Results window after orders have been resolved (adjudicated). Adjudication results are also available through the Reports menu.

Default save game directory

Select the default directory for save game files.

Clear recently used file list

Clears the files (if any are listed) in the most-recently-used file list, which is located in the File menu. Note that this is not undoable.

Display

Order Format

Determines how orders are formatted within the order list. An example order is shown so you can see how the formatting options affect how orders are displayed.

Export

The export settings affect how PNG and JPEG maps are exported. It does not affect PDF and SVG exported maps.

Image Export Size

Exported map size may be scaled to the desired size.

Image Export Settings

JPEG Image Quality

Higher numbers are higher quality, and larger files. Lower numbers are lower quality and smaller files. Generally, a good compromise is in the 70-80 range.

Maximum Colors in PNG Images

PNG (Portable Network Graphics) files may be exported as 1, 2, 4, 8, or 32 bits-per-pixel (bpp) images. This corresponds to 2, 4, 16, 256, and "unlimited" colors, respectively. Images with fewer colors (fewer bits per pixel) will be smaller but may have decreased quality.
