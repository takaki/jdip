Importing Games from floc.net

What is floc.net?

www.floc.net is a website created by Alain Tesio that allows games played on email Judges to be tracked. Registered Judge games may be searched and tracked. Entire game histories and maps are also available.

For more information about online Judges, see www.diplom.org.

Importing a Game from floc.net

jDip is able to directly import game histories from floc.net, creating an entire game in jDip format that may be viewed and adjudicated. To do this, the exact game name and judge name must be known.

Only variants supported by jDip may be imported.

Searching floc.net

The ability to search floc.net is not currently available in jDip. However, floc.net's extensive search capabilities may be used by browsing to http://www.floc.net.
