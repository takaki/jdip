Edit Menu

Undo and Redo actions, delete and select orders here. Edit Mode may also be enabled and disabled via the Edit menu, to add or remove units from the map. Preferences can be set.

Persistent Undo/Redo

Undo and Redo are persistent across saved games. This includes order resolutions, order additions, and edit-mode changes. Even several turns later in a game you opened from weeks ago, you can undo all the actions that have occured since the beginning of a game!
